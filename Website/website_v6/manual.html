The Manual

What is a cellular automata?

A cellular Automata is the plural form of cellular automaton. A cellular automaton (CA) is a model of a system of "cell" objects with the following characteristics:

Grid:
The cells live on a grid.

State:
Each cell has a state.

Neighborhood:
This can be defined in any number of ways, but it is typically a list of adjacent cells.

How to operate our simulator website

Set #Rows, #Cols, and CellWidth
This section allows you to specify the size of the grid and how large you would like your cells to be.
The #Rows and #Cols allows you to specify the columns and rows that go into your grid.
The Cellwidth allows you specify the sizing of each cell within the grid.

Set Delay
Set delay specifies how much time there is between updates of cells.
The shorter the delay, the faster the grid updates.

Set InitBoard function
The InitBoard function is the "initial board function".
This allows you to set what the initial board will look like.

Set UpdateCell function
The UpdateCell function updates the cells.
This defines how the cells interact with each other.

Set DrawBoard Function
The set DrawBoard Function allows you to specify the colors of the state of the cell


Preset List
The preset list is a list of preset CA's that you can click through and see.

Change colors
The bottom box is set there to allow you to change the colors of the grid.
The numbers each specify a certain state.
For the colors, just enter the hex representation coding in the format #XXXXXX
